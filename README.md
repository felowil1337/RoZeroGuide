# Ragnarok Zero Guide (Deutsch)

Statische Website als Vorbereitung auf den Ragnarok Zero Global Release.

Nach Push auf GitLab wird automatisch via GitLab Pages deployed:
`https://<username>.gitlab.io/<reponame>/`

## Struktur

- `index.html` — Startseite
- `klassen.html` — Klassen (1./2. Job)
- `leveling.html` — Level-Routen, Episode-One-Walkthrough, EXP-Booster
- `instanzen.html` — Memorial-Dungeons (Pre-Trans)
- `fever.html` — Fever-Felder + Equipment mit Stats
- `auto-hunt.html` — Auto-Hunt + Auto-Pot System
- `trans-klassen.html`, `high-instanzen.html`, `endgame-*` — Zukunftsthemen
- `style.css` — Shared Stylesheet
