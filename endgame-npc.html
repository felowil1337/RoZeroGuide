🔄 Empfohlener Endgame-Flow

  1. Post-Rebirth Trans → erste Runs in OGH + Lv 110–130-Dungeons
  2. Temporal Crystals farmen → Temporal Boots für deine Klasse
  3. Memorial-MD-Crafting fortsetzen → Expedition → Guardian → Conqueror (braucht Crimson Crystal + Holy Stone)
  4. Mora Coins → 3rd-Job-Set für deine Trans-Class (über Hazy Forest Soul-Rescue Dailies)
  5. Class C/B/A Coins → Malangdo-Enchants für Lv 3-4-Weapons
  6. Mythril Ore → Hard-Mode-MDs + Fever-Maps + Abyss Lake Cave 4
  7. Splendide/Manuk → Buff-Consumables für MVP-Runs
  8. Para Market (Fri/Sat/Sun) → seltene Crafting-Mats
  9. Etel Materials → Stern-Grade-System (erst ab Lv 200+)
